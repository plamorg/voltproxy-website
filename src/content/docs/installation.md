### Build from source

Ensure you have [Go](https://go.dev/doc/install) 1.21 or newer.

1. Clone the repository:

```sh
$ git clone "https://github.com/plamorg/voltproxy.git"
```

2. Build voltproxy:

```sh
$ cd voltproxy/
$ go build
```
